# Tuto React query

## Notions vu dans ce projet

- Typescript
- React query
- React hook form
- Storybook

- Context react
- Custom hook

## Lancer le projet

1. Lancer json-server

```bash
# A la racine du projet

json-server --watch db.json

# Serveur lancer sur le port 3000
```

2. Lancer le serveur de dev react

```bash
pnpm run dev

# Serveur lancer sur le port 3001
```

## Storybook

```bash

npm run storybook

# Serveur lancer sur le port 3000
```
